# Vue2-Sub-App

A plugin can make any component can be used as the Root-like component.

## Install

```shell
npm install vue2-sub-app
```

## Usage

```js
// main.js
import Vue from 'vue'
import Vue2SubApp from 'vue2-sub-app'

Vue.use(Vue2SubApp)

new Vue({
  /* ... */
})
```
